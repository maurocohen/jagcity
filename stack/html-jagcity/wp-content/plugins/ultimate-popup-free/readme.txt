== Description ==

This plugin will enable AWESOME popup in your WordPress website.

<h2><a href="http://demos.ultimatepopup.com/?theme=Popup%20will%20show%201%20time%20per%20day">See Live Demo Here</a></h2>
		
You can add unlimited popup with its configuration. You can change popup content, title, popup codes &amp; cookie expiry. We added effective settings panel for each popup.
